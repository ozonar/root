# Сайт

Таск-менеджер в виде древовидного списка чекбоксов с WYSIWYG-редактированием. Проекты группируют страницы, страницы содержат задачи-чекбоксы.
Статусы задач настраиваются для каждого проекта (базовые: finished, processed). Авторизация по email+пароль с токенами. На странице — полноценное редактирование с drag-n-drop, контекстными меню, назначением исполнителей, приоритетами.

# Сущности

- **`User`** — пользователь (email, пароль с солью, роли, токены, currentProject)
- **`Project`** — проект-контейнер (название, владелец, создатель)
- **`UserProject`** — связь пользователя с проектом (роль: OWNER и др.)
- **`Page`** — страница (заголовок, версия, editedAt, deletedAt — soft delete)
- **`Task`** — задача-чекбокс (текст, описание, order, parent_id — вложенность, assignee, статус, isPriority)
- **`Status`** — статус задачи (systemName, name, icon, active, привязка к проекту)
